FeedBack

The portfolio design is clean and user-friendly. It is evident that a lot of effort went into crafting an interface that balances aesthetics with functionality. The choice of colors and typography complements the overall theme perfectly.

The responsiveness of the website is good, but there are a few glitches when viewing on very small screens like older iPhones. It would be great if these could be fixed to ensure the site works flawlessly on all devices.

The projects section is well-organized, but I feel that adding a filter option to sort projects by categories or technologies would improve the user experience. For instance, users could view React projects separately from data science projects.

The animations on the website add a nice touch and make the navigation feel more dynamic. However, some animations are slightly too long, which might make the user experience feel a bit slow for visitors in a hurry.

I appreciate the inclusion of a light/dark mode toggle. However, it would be great if the preference was saved in cookies or local storage so that the user's choice persists across sessions.
